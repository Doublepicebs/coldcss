Doublepicebs Design System

What is COLD?

COLD, stands for Customisable, Operating and Leading Design. It's the Doublepicebs Design System for Web.

Although COLD was meant and optimised for internal work, it is available for the public; and thus we wanted it to be highly customisable, using CSS variables, without preprocessors, exactly like Shoelace. (see below)

What is COLD based on?

COLD relies heavily on the core ideas of Shoelace css and even many basic styles and components. COLD works without cssnext, unlike Shoelace. It also ports only the necessary components and styles, with the small version of the Doublepicebs color palette.

For the grid system, we use Simplegrid.
