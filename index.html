Doublepicebs Design System

What is COLD?

COLD, stands for Customisable, Operating and Leading Design is the Doublepicebs Design System for Web.

Although COLD was meant and optimised for internal work, it is available for the public; and thus we wanted it to be highly customisable, using CSS variables, without preprocessors, exactly like Shoelace.

The docs

Features


Completely Customisable

COLD is meant to be easy, and customisable, with A LOT of css variables (css custom properties). No preprocessors are required.


Open source

The source is available for free on github, feel free to get it, edit it and use it as you wish.

Responsive

Our stylesheets are beautiful, responsive and more importantly, mobile first.

What is COLD based on?

COLD relies heavily on the core ideas of Shoelace css and even many basic styles and components. However, COLD works without cssnext as it was meant for static site development. It also ports only the necessary components and styles, with the small version of the Doublepicebs color palette.

For the grid system, we use Simplegrid.
